Preamble

History of the World Wide Web and mission of W3C.

History

  • In 1989 Tim Berners-Lee proposed the Mesh
  • at CERN, in Geneva, Switzerland.
  • Renamed the Mesh to World Wide Web in 1990.

The Proposal

Proposal for an "information management system" which is "Vague, but exciting".

Sir Tim Berners-Lee

Tim Berners-Lee created the first Web page, Web browser, and Web server.

He also wrote the first specifications for URLs, HTTP, and HTML.

World Wide Web Consortium (W3C)

In 1994, Tim Berners-Lee founded the W3C.

W3C's primary activity is to develop protocols and guidelines (Web standards) that ensure long-term growth for the Web.

For example: HTML and CSS.

Basic HTML

Encoding, elements, tags, attributes, ….

HTML5

Umbrella term for:

  • Hypertext Markup Language version 5 (HTML5)
    Holds and organizes content into a logical structure
  • Cascading Style Sheets version 3 (CSS3)
    Takes care of presentation and style of HTML documents.
  • JavaScript
    Allows actions and turns documents into potential Web applications.

Markup

Annotating a document with extra information, which is (e.g. in the Browser) omitted from users.

The HTML specification prescribes how to markup semantics (meaning).

<!DOCTYPE html>
<html>
  <head>
    <title>Sample page</title>
  </head>
  <body>
    <h1>Sample page</h1>
    <p>This is a <a href="demo.html">demo</a>.</p>
    <!-- this is a comment -->
  </body>
</html>

Doctype

The document type declaration (DTD) is the formal definition of a particular HTML version in the very first line of an HTML document.

<!DOCTYPE html>

Root Element

The root element is usually the <html> element, it can be found at the root (of the tree of elements) of HTML documents.

<html>…</html>

Head Element

The <head> element, a container for metadata (data about data), is the first element inside the <html> element.

<head>…</head>

Title Element

The <title> element represents the title or name of an HTML document and should identify its content in a fairly wide context and about 56 characters.

<title>…</title>

Body Element

The <body> element 

<body>…</body>

Tags & Elements

Each* element is denoted in the source by a start tag <> and an end tag </>.

<element>…</element>

Nesting

Tags have to be nested such that elements are all completely within each other, without overlapping.

<p>This is <em>very <strong>wrong</em>!</strong></p>
<p>This <em>is <strong>correct</strong>.</em></p>
