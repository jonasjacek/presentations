Was sind Standards?

Ein Überblick über Standards im Internet.

Unterscheidungen

  • Internetstandards
  • Webstandards
  • Best (Current) Practices

Internetstandards

Technische Spezifikation, die ...

  • Funktionsweisen des Internet beschreiben
  • einen sehr hohen Reifegrad haben
  • von einer breiten Öffentlichkeit unterstützt werden.

Beispiele für Internetstandards

  • Internet Protocol (IP)
  • Domain Name System (DNS)
  • Hypertext Transfer Protocol (HTTP)
  • File Transfer Protocol (FTP)
  • ...

Webstandards

Technische Spezifikation, die ...

  • Funktionsweisen des World Wide Web beschreiben
  • einen sehr hohen Reifegrad haben
  • von einer breiten Öffentlichkeit unterstützt werden.

Beispiele für Webstandards

  • Hypertext Markup Language (HTML)
  • Cascading Syle Sheets (CSS)
  • Web Open Font Format (WOFF)
  • Webmention (Webmention)
  • ...

Best (Current) Practices

Sind technische Spezifikation, die ...

  • flexibler sind als ein Standard
  • sich noch entwickeln
  • ein beachtliches Leistungsniveau erreicht haben.

Beispiele für Best (Current) Practices

  • Best Current Practices (bcp1)
  • The Internet Standards Process (bcp9)
  • Tags for Identifying Languages (bcp47)
  • Randomness Requirements for Security (bcp106)
  • ...

Wer Macht Standards?

Informationen zu Normungsgremien im Internet.

Normungsgremien

  • Internet Engineering Task Force (IETF)
  • Internet Assigned Numbers Authority (IANA)
  • World Wide Web Consortium (W3C)
  • International Organization for Standardization (ISO)
  • etc.

Internet Engineering Task Force (IETF)

Webseite https://ietf.org/
Standards TCP/IP, IPv6, HTTP, DNS, etc.
Mitglieder 80.000+ Personen
Gegründet: 1986

Internet Assigned Numbers Authority (IANA)

Webseite https://www.iana.org/
Betreibt IP Adressen, DNS Root Zone, etc.
Betreiber Internet Corporation For Assigned Names and Numbers (ICANN)
Gegründet: 1988

World Wide Web Consortium (W3C)

Webseite https://www.w3.org/
Standards HTML, XML, CSS, SVG, WCAG, etc.
Mitglieder 451 Organisationen
Gegründet: 1994

International Organization for Standardization (ISO)

Webseite https://www.iso.org/
Standards WCAG, MathML, ISO 639, ISO 3166, etc.
Mitglieder 162 Länder
Gegründet: 1947

Gemeinsamkeiten

  • Breit aufgestellt
    Regierungsvertreter, Universitäten, Privatunternehmen, Einzelpersonen
  • Offene Prozesse
    Größtenteils transparente Prozesse, Beteiligung und Feedback sind möglich
  • Freie Nutzung
    Standards sind frei zugänglich, möglichst weite Verbreitung angestrebt

Vorteile Durch Standards?

Vorteile durch die Nutzung von Standards.

Um die Bedeutung von Webstandards zu erkennen lohnt sich ein Blick in die Vergangenheit des Internet, in eine Zeit vor Webstandards.

Standards haben Einfluss auf die Interoperabilität, die Zugänglichkeit und die Benutzerfreundlichkeit von Webseiten und anderen Web-Produkten.
