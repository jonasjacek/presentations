Wer Macht Standards?

Normungsgremien

  • Internet Engineering Task Force (IETF)
  • Internet Assigned Numbers Authority (IANA)
  • World Wide Web Consortium (W3C)
  • International Organization for Standardization (ISO)
  • etc.

Internet Engineering Task Force (IETF)

Webseite https://ietf.org/
Standards TCP/IP, IPv6, HTTP, DNS, etc.
Mitglieder 80.000+ Personen
Gegründet: 1986

Internet Assigned Numbers Authority (IANA)

Webseite https://www.iana.org/
Betreibt IP Adressen, DNS Root Zone, etc.
Betreiber Internet Corporation For Assigned Names and Numbers (ICANN)
Gegründet: 1988

World Wide Web Consortium (W3C)

Webseite https://www.w3.org/
Standards HTML, XML, CSS, SVG, WCAG, etc.
Mitglieder 451 Organisationen
Gegründet: 1994

International Organization for Standardization (ISO)

Webseite https://www.iso.org/
Standards WCAG, MathML, ISO 639, ISO 3166, etc.
Mitglieder 162 Länder
Gegründet: 1947

Gemeinsamkeiten

  • Breit aufgestellt
    Regierungsvertreter, Universitäten, Privatunternehmen, Einzelpersonen
  • Offene Prozesse
    Größtenteils transparente Prozesse, Beteiligung und Feedback sind möglich
  • Freie Nutzung
    Standards sind frei zugänglich, möglichst weite Verbreitung angestrebt

Was sind Vorteile der Nutzung von Standards?

Bedeutung für User Experience, Design, Marketing.

Standards haben Einfluss auf die Interoperabilität, die Zugänglichkeit und die Benutzerfreundlichkeit von Webseiten und anderen Web-Produkten.
